Real-Time Chat Application Backend

Developed the backend system for a real-time chat application using Python and FastAPI. Implemented real-time communication features using WebSockets and created REST APIs for user interaction.

Features

- Real-time messaging using WebSockets
- REST API development using FastAPI
- JWT authentication for secure access
- Database integration using MySQL
- User login and registration APIs
  
Technologies Used

- Python
- FastAPI
- WebSockets
- MySQL
- JWT Authentication
  
  Backend Responsibilities

- Developed backend APIs
- Implemented authentication and authorization
- Managed database connectivity
- Handled real-time communication logic
